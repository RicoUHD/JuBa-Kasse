Ersteinrichtung

Willkommen! Bitte konfigurieren Sie die Anwendung.

1. Allgemeine Informationen

Dieser Name wird in der App und in E-Mails angezeigt.

2. Firebase Frontend-Konfiguration

Kopieren Sie das JSON-Objekt aus Ihren Firebase-Projekteinstellungen (Web-App).

3. Firebase Service Account (Backend)

Generieren Sie einen neuen privaten Schlüssel in den Firebase Projekteinstellungen unter "Dienstkonten" und fügen Sie den gesamten Dateiinhalt hier ein.

4. E-Mail Einstellungen (Optional)

Diese Einstellungen werden für Benachrichtigungen (Anfragen, Status-Mails) benötigt. Sie können diesen Schritt überspringen.

Bei Gmail wird dringend ein App-Passwort empfohlen.
